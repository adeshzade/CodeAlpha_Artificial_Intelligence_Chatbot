# 🤖 Artificial Intelligence Chatbot

An AI-powered chatbot developed in **Java** that enables interactive communication using **Natural Language Processing (NLP)** concepts and a **rule-based response system**. The chatbot can understand user queries, answer frequently asked questions, and provide real-time responses through a simple Java Swing graphical user interface.

---

## 📌 Project Overview

The **Artificial Intelligence Chatbot** is a desktop-based application built using Java to demonstrate the fundamentals of Artificial Intelligence, Natural Language Processing, and Object-Oriented Programming. The chatbot processes user input, identifies keywords, and generates appropriate responses based on predefined rules.

This project is designed for students, beginners, and developers who want to learn chatbot development using Java without relying on external AI services. The modular architecture allows future integration with Machine Learning models and cloud-based AI APIs.

## ⚙️ How It Works

1. User enters a message.
2. The chatbot processes the input.
3. Keywords are extracted using basic NLP techniques.
4. The rule-based engine searches for matching responses.
5. If a match is found, an appropriate answer is returned.
6. Otherwise, the chatbot displays a default response.
7. The conversation continues until the user exits.
